Declaration:

Three types of declaration:
  1. var(variable)
  2. let
  3. constant(const)

Data types:

  1. int(integer i.e. 253,564,12 etc
  2. char(character i.e. letter or any number)
  3. float(i.e. 24.5,67.6)
  4. string(a word is string)
  5. boolean( it shows only 2 value true or false)
  6. array(i.e. [ a b c d e f ] It is type of object

Hoisting:

1.Hoisting is the behavior of moving all the declarations to the top of the scope before code execution.
2. Hoisting is not applied for Let and Constant kind of datatype.

Types of errors:

  1. Type error(The TypeError happens if a variable is of an unexpected type or access to a nonexistent method. )
  2. Syntax error
  3. Reference error(ReferenceError. The ReferenceError raises when you reference a variable, a function, or an object that does not exist.)
